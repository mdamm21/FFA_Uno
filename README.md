## Fischfutterautomat

Ziel ist die Entwicklung eines Fischfutterautomat, welcher einmal täglich, alle Sorten von Fischfutter füttern kann.

## Verwendete Hardware

   - Arduino Uno R3
   - D1 Board NodeMCU ESP8266MOD-12F
   - MG996R Micro Digital Servo Motor
   - HC-SR04 Ultraschallsensor
   - 9V Netzteil
   - Breadboard
   - Jumpercable
   - 2x 1kΩ Widerstand
